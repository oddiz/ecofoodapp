This app finds best possible food combination that gives the highest skill point per day. This is achieved by generating random diets with given inputs, calculating the skill points and selecting the best diet.

Diet Scale input determines the diet size for test. Each diet will be filled with randomly selected foods from Selected Foods list.

Simulation Scale input determines the number of random diets to generate.

Calculate button calculates SPs of all generated diets and outputs the highest yield.

After a calculation has been made. The output is then stored in highscore database, which will be compared with all previous results.

Highscores are categorised by Diet Scale. Also you can see the best diet accross all categories on page load.

NEW: Now you can also specify prices for food and set a budget limit to generate results less or equal than your specified budget. The prices are saved automatically as cookies so you don't have to write them every time.

SP calculation is made as follows:

[
( N1 × C1 ) + ( N2 × C2 ) + ( N3 × C3 )

——————————————————————

C1 + C2 + C3
 		] × Balance Modifier × Skill Gain Multiplier + Base Skill Gain = SP/Day
  • "N" represents total nutrients of each food. Simply add up all 4 nutrient values. N1 is first food, N2 is second food, etc.
  • "C" represents calories of each food. C1 is first food, C2 is second food, etc.
  • Balance modifier ranges from 0.5 to 2, with 0.5 being no balance, and 2 being a perfect balance.
  • Skill gain multiplier is configurable in the server difficulty settings. It has different default values depending on the collaboration preset.
  • Base skill gain can be modified in EcoSim.eco. It defaults to 12.

Source: https://eco.gamepedia.com/Skill_Points
